忍法（鞍馬神流・流派忍法）

霧雨（きりさめ）
タイプ：攻撃忍法 / 間合：1 / コスト：1
指定特技：水術、走法
接近戦攻撃。攻撃が成功すると、目標に接近戦ダメージを1点与える。その後、任意の効果を選び、それを無効にすることができる（その効果が累積する場合、累積した効果すべてで無効にする）。
青嵐（せいらん）
タイプ：攻撃忍法 / 間合：2 / コスト：2
指定特技：走法
接近戦攻撃。攻撃が成功すると、目標に接近戦ダメージを1点与える。目標のプロット値に関係なく「間合2」以上だった場合、さらに接近戦ダメージを1点与えることができる。
五月雨（さみだれ）
タイプ：攻撃忍法 / 間合：1 / コスト：2
指定特技：刀術
射撃戦攻撃。攻撃が成功すると、目標に射撃戦ダメージを1点与える。使用可能な攻撃忍法があれば、その忍法を使って、攻撃をもう一度行うことができる。
射射（しゃしゃ）
タイプ：攻撃忍法 / 間合：2 / コスト：2
指定特技：弓術
射撃戦攻撃。攻撃が成功すると、目標に射撃戦ダメージを1点与える。自分のプロット値を1上昇する。
前腕（ぜんわん）
タイプ：攻撃忍法 / 間合：1 / コスト：1
指定特技：体術
接近戦攻撃。攻撃が成功すると、目標に接近戦ダメージを1点与える。目標が「回避判定」に成功しても、さらに接近戦ダメージを1点与えることができる。
抜討（ぬきうち）
タイプ：サポート忍法 / 間合：なし / コスト：1
指定特技：なし
自分よりプロット値が高いキャラクターから攻撃を受けるときに使用できる。その攻撃に対する回避判定にプラス2の修正がつく。その判定に成功した場合、使用可能な攻撃忍法があれば、その忍法を使って、攻撃を一度行うことができる。
天狼（てんろう）
タイプ：サポート忍法 / 間合：なし / コスト：2
指定特技：骨法術
自分の命中判定の前に使用できる。指定特技の判定に成功すると、その攻撃の命中判定のファンブル値は、自分のプロット値と同じ値になる。
影縫（かげぬい）
タイプ：サポート忍法 / 間合：なし / コスト：なし
指定特技：なし
プロット公開時、自分のプロットを行動順に変更できる。戦闘に参加しているキャラクターを目標に選ぶ。目標のプロット値が自分のプロット値よりも大きい場合、自分のプロット値を目標のプロット値と同じ値に変更する。この忍法は一度の戦闘に一度まで使用できる。
返討（かえりうち）
タイプ：サポート忍法 / 間合：なし / コスト：なし
指定特技：気配
ドラマシーン中、誰かが忍法の目標に自分が選ばれたときに使用できる。その忍法の使用者一人を目標に選ぶ。指定特技の判定に成功すると、目標に接近戦ダメージを1点与える。この効果は、戦闘のシーンに登場していなくても使用できる。
俄雨（にわかあめ）
タイプ：サポート忍法 / 間合：1 / コスト：1
指定特技：意気
戦闘中、自分の【生命力】が1点以上失われたときに使用できる。この忍法の間合内にいるキャラクターをすべて対象とする。指定特技の判定に成功すると、そのラウンドの間、自分の攻撃に対して目標が行う回避判定にマイナス1の修正がつき、そのラウンドの間、目標は汎用のサポート忍法を使用できなくなる。
透視（すかし）
タイプ：装備忍法 / 間合：なし / コスト：なし
指定特技：なし
自分の回避判定、もしくは奥義破り判定のサイコロを振る直前に使用できる。【生命力】を1点消費すると、その判定にプラス2の修正がつく。
